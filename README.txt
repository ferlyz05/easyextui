EASYEXTUI
Alternatif ExtUI untuk jEasyUi

UI dibuat semirip mungkin dengan Ext untuk itu js library banyak di lakukan modifikasi,
menggunakan ini berarti menggunakan library yang di ambil dari tempat ini

VER 1.0.3 - 15-2-2014
- Perubahan core, core dasar diambil dari: http://www.jeasyui.com/easyui/jquery.easyui.min.js
- Penambahan atribute frame pada panel, memodifikasi panel, panel-header, panel-body dengan class panel-frame, panel-frame-header, panel-frame-body
- Penmbahan Input type checkbox dan radio menggunakan html5, penggunaan seperti biasa <input type="radio/checkbox" title="Foo"> hasil: label.radio > input + span.radio-buton + span{title}
- Penambahan auto width form widget
- Penambahan library toolbar, penggunaan #{prefix}-toolbar attribute position="{value}" {position[value]}: top right bottom left, plain={value} {plain[value]}: true/false
- Penambahan library buttongroup penggunaan #{prefix}-buttongroup > x-linkbutton
